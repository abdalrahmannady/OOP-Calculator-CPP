# OOP Calculator in C++

A simple object-oriented calculator project built using C++.

## Features
- Add
- Subtract
- Multiply
- Divide
- Clear
- Enum-based operation tracking
- Operation history tracking

## Concepts Used
- Object-Oriented Programming (OOP)
- Encapsulation
- Enums
- State management

## Example Output
Result after Adding 100 is : 100
Result after Subtracting 20 is : 80
Result after Multiplying 2 is : 160
